# Trident Risk & Advisory Website

This is a Vite React website prepared for deployment on Vercel.

## Local preview

```bash
npm install
npm run dev
```

## Vercel settings

Framework Preset: Vite
Build Command: npm run build
Output Directory: dist
Install Command: npm install

## Logo

Replace `public/logo.svg` with your official logo if required. If you use a PNG logo, place it in `public/logo.png` and update the image path in `src/App.jsx`.
